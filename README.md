# SparkCalc

SparkCalc is a mobile-first electrical maths and study app for apprentices. It shows answers, formulas, unit conversions, step-by-step working, and warnings when inputs look missing, zero, negative, or unrealistic.

## Run It

```bash
npm install
npm run dev
```

## Test It

```bash
npm test
```

## Included Calculators

- Ohm's Law
- Single-phase power
- Series and parallel resistance
- Three-phase power
- Voltage drop training calculator
- Fault current
- Energy cost
- Unit converter

The app also includes a searchable formula sheet, random practice questions with local score tracking, saved calculations using localStorage, theme settings, decimal-place settings, and default voltage settings.

## Safety Disclaimer

Training aid only. Always confirm real electrical work against current standards, site requirements, and a licensed electrician.

For real cable sizing, confirm against the current wiring rules and cable tables. SparkCalc does not replace AS/NZS standards, cable tables, site rules, or a licensed electrician.
